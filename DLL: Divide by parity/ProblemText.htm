You have been given a DLL with N nodes, where each node contains one Integer. You are supposed to split the elements of this list into two other lists. The first one should contain every node having an odd element from the original list, and the second should contain every node having an even element.

In the first line of the input you will find an integer, representing the number of nodes, and in the second line you will find the elements of the list with a single whitespace between them. In the first line of the output, you should print the list containing the odd elements and in the second you should print the list containing the even elements.

Name of the class (Java): DivideOddEven
